# 🚀 AI PDF to MCQ Generator

An AI-powered web application that converts PDFs into intelligent quizzes and study material using Gemini AI.

## 🌐 Live Demo

Frontend (Vercel):
https://dl-project-pdf-to-mcq.vercel.app

Backend (Render):
https://dl-project-pdf-to-mcq.onrender.com

---

## ✨ Features

* 📄 Upload PDFs and generate AI-powered MCQs
* 🧠 Multiple question types:

  * Multiple Choice
  * True / False
  * Fill in the Blanks
* 🎯 Difficulty levels:

  * Easy
  * Medium
  * Hard
* ⚡ Real-time MCQ generation progress
* 🔄 Async processing pipeline
* 🔐 JWT Authentication System
* 📚 Detailed answer explanations
* 📦 Large PDF support with chunked processing
* 🚀 Optimized concurrency for faster generation
* ☁️ Production deployment with monitoring

## ⚙️ How It Works

1. Upload PDF
2. Extract and chunk text
3. Process chunks asynchronously
4. Generate MCQs using Gemini AI
5. Save and display generated quizzes
