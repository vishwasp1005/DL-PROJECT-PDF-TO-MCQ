# AI PDF to MCQ Generator

## 🚀 Live Demo
https://dl-project-pdf-to-mcq.vercel.app

## 📌 Features
- Generate MCQs from PDFs using LLaMA 3
- Flashcard-based learning UI
- JWT authentication system

## 🛠 Tech Stack
- FastAPI, React, SQLite

## ⚙️ How it works
- Upload PDF → Extract text → Generate questions using LLM
